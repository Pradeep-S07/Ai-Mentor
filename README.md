Skill Navigator Pro
Project Information

Skill Navigator Pro is a web application that helps users explore technical skills, understand sub-skills, and follow structured learning paths. The platform is designed to guide students and developers toward becoming industry-ready through curated roadmaps and micro-learning.

How to Edit This Project

There are several ways to edit and run this application locally.

Prerequisites

Node.js (v18 or later recommended)

npm (comes with Node.js)

Option 1: Use Your Preferred IDE (Recommended)
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev

Option 2: Edit Directly on GitHub

Open the file you want to edit.

Click the ✏️ Edit button.

Make your changes and commit them.

Option 3: Use GitHub Codespaces

Go to the repository main page.

Click Code → Codespaces → New codespace.

Edit files inside the browser IDE.

Commit and push your changes when finished.
